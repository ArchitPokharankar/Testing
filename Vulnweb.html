Practical 1 — Cryptography (Symmetric, Asymmetric, Hashing)
Tools: CryptoTool, HashCalc, HashMyFiles
  1. Open CryptoTool > Choose Caesar Cipher. Enter key (e.g., 3). Encrypt/Decrypt to observe.
  2. Vigenère: CryptoTool > Vigenère. Enter keyword (e.g., KEY). Encrypt/Decrypt.
  3. RSA: CryptoTool > RSA > Generate keys. Encrypt with public key; decrypt with private key.
  4. Hashing: Use HashCalc/HashMyFiles/CryptoTool to compute MD5, SHA1, SHA256 for text or files.
Below: example commands & minimal scripts that illustrate hashing and RSA with Python (for study only).
## Example: Compute MD5 / SHA256 hashes in Python
import hashlib
text = b"hello world"
md5 = hashlib.md5(text).hexdigest()
sha256 = hashlib.sha256(text).hexdigest()
print('MD5:', md5)
print('SHA256:', sha256)

## Example: RSA using PyCryptodome (install with pip install pycryptodome)
from Crypto.PublicKey import RSA
from Crypto.Cipher import PKCS1_OAEP
key = RSA.generate(2048)
priv = key.export_key()
pub = key.publickey().export_key()
# Encrypt
cipher = PKCS1_OAEP.new(RSA.import_key(pub))
ciphertext = cipher.encrypt(b"Secret message")
# Decrypt
decipher = PKCS1_OAEP.new(RSA.import_key(priv))
print(decipher.decrypt(ciphertext))
Practical 2 — Hashcat (Crack MD5 & SHA-256)
Tools: Hashcat (download .7z). Use Windows CMD or PowerShell.
  1. Download Hashcat (7z). Extract to a folder.
  2. Create abc.txt containing the hash and wordlist.txt with candidate passwords.
  3. Move both files into Hashcat folder. Open CMD & cd into that folder.
  4. Run: 
    hashcat.exe -m 0 -a 0 abc.txt wordlist.txt   # MD5
hashcat.exe -m 1400 -a 0 abc.txt wordlist.txt # SHA256
  5. If the password is in the wordlist, Hashcat will recover it.
# Example: Make a simple wordlist (Linux/Mac/WSL/Cygwin)
echo -e "password
123456
letmein
admin" > wordlist.txt

# Example: get md5 of string (PowerShell)
"hello" | Out-File -Encoding ASCII tmp.txt; Get-FileHash tmp.txt -Algorithm MD5

Practical 4 — Cryptanalysis using CryptoTool
Open CryptoTool and perform encrypt/decrypt for each listed cipher.
  1. Use CryptoTool: Caesar, Vigenère, Playfair — encrypt and decrypt sample plaintexts.
  2. DES (ECB), AES (CBC), RC4: use example keys and modes; compare ciphertexts.
  3. For RC4 brute-force: use CryptoTool's entropy/brute options as described in the PDF.
# Example: AES-CBC with PyCryptodome (study only)
from Crypto.Cipher import AES
from Crypto.Util.Padding import pad, unpad
key = b"16bytecipherkey!"
iv = b"16bytesivvector"
cipher = AES.new(key, AES.MODE_CBC, iv)
ciphertext = cipher.encrypt(pad(b"Secret Message", 16))
print(ciphertext.hex())
# Decrypt
plain = unpad(AES.new(key, AES.MODE_CBC, iv).decrypt(ciphertext), 16)
print(plain)

Practical 8 — Scanning using Nmap
Commands below can be run in terminal (install nmap).
  1. Intense Scan / Quick Scan via Nmap GUI or: 
    nmap -T4 -A -v target.example.com   # intense
nmap -T4 -F target.example.com          # quick
  2. SYN scan: nmap -sS target
  3. ACK scan: nmap -sA target
  4. FIN: nmap -sF target | NULL: nmap -sN target | XMAS: nmap -sX target
nmap -sS www.google.com
nmap -sA www.google.com
nmap -sF www.google.com
nmap -sN www.google.com
nmap -sX www.google.com
Practical 9 — Wireshark (Sniffer)
Capture traffic, apply filters, and interpret HTTP responses.
  1. Start Wireshark capture on the correct adapter.
  2. Use filters such as: ip.addr==192.168.0.107, http, http.response.code==404.
  3. To inspect browser input on a vulnerable test site use http filter while performing the action.
  4. To test client-server sniffing, run the provided Python server/client on two machines and capture the traffic.
# Simple server (Python)
import socket
s = socket.socket()
port = 8080
s.bind(('0.0.0.0', port))
s.listen(5)
print('Server listening')
while True:
    c, addr = s.accept()
    print('Got connection from', addr)
    c.sendall('Thank you for connecting!'.encode())
    c.close()

# Simple client (Python)
import socket
s = socket.socket()
s.connect(('192.168.2.60', 8080))
print(s.recv(1024).decode())

Practical 12 — File Hiding / Stealth Files (Windows)
Simple OS techniques to hide or disguise files.
  1. Hide file with attrib: attrib +h +s demo.txt. To unhide: attrib -h -s demo.txt.
  2. Rename to disguise type (change extension) or create double extensions to confuse users.

attrib -h -s demo.txt
# Example: rename to disguise
